# Weather App (CLI)

Консольное приложение на Python для получения текущей погоды
по выбранной стране и городу.

## Как работает
- пользователь выбирает страну
- выбирает город
- приложение показывает текущую погоду

## Используемые технологии
- Python
- requests
- Open-Meteo API
